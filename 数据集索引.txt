# 输电铁塔AI巡检 · 公开数据集索引
====================================

## 1. InsPLAD（144 stars ⭐）
- 地址：https://github.com/andreluizbvs/InsPLAD
- 内容：输电线路资产巡检，绝缘子/金具/防振锤等多类
- 数量：多类混合
- 格式：图片分类/检测
- 下载：git clone https://github.com/andreluizbvs/InsPLAD.git
        或者 GitHub 页面点 Code → Download ZIP

## 2. FINet
- 地址：https://github.com/zhangzhengde0225/FINet
- 内容：输电线路绝缘子缺陷检测
- 格式：检测数据集

## 3. TLID-Dataset
- 地址：https://github.com/Caughyhzd/TLID-Dataset
- 内容：输电线路绝缘子缺陷检测 Towards Defect Detection of Transmission Line Insulator
- 格式：检测数据集

## 4. 绝缘子检测数据集（中文）
- GitHub 搜索：insulator detection dataset
- 搜索结果中有多个中文项目，如：
  - "yolo绝缘子目标检测-电力绝缘子设备检测(代码+数据集)"
  - "绝缘子表面污染检测数据集 (ISCD)"

## 5. 鸟巢检测数据集
- 搜索："bird nest detection transmission line"
- GitHub 上鸟巢数据集的专用公开仓库较少，可关注：
  - 通用目标检测数据集中包含鸟巢类别的

## 下载方式
每个仓库页面：
1. 点击绿色 Code 按钮 → Download ZIP
2. 或 git clone <仓库地址>
3. 解压后放入 D:\transmission_dataset\raw\ 目录

## 目录结构（放好后长这样）
D:\transmission_dataset\raw\
├── image001.jpg
├── image001.txt  (标注)
├── image002.jpg
├── image002.txt
└── ...
